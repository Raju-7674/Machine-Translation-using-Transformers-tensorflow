# 🌐 Machine Translation using Transformer (TensorFlow)

## 📌 Overview

This project implements a Neural Machine Translation (NMT) model using the Transformer architecture in TensorFlow. The model translates English sentences into Hindi using an Encoder-Decoder architecture with Multi-Head Attention, Positional Encoding, and Feed Forward Networks.

---

## 🚀 Features

- English to Hindi Machine Translation
- Transformer Encoder-Decoder Architecture
- Multi-Head Self-Attention
- Positional Encoding
- Feed Forward Neural Networks
- Padding Mask & Look-Ahead Mask
- Tokenization and Sequence Padding
- TensorFlow Implementation

---

## 🛠️ Technologies Used

- Python
- TensorFlow
- NumPy
- Pandas
- Google Colab

## 📂 Dataset

Parallel English-Hindi translation dataset:

- `train.en` - English sentences
- `train.hi` - Hindi translations
