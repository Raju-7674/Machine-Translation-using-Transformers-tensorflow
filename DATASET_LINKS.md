# 📂 Dataset

The dataset used in this project is hosted externally due to its size. It contains parallel English-Hindi sentence pairs used to train and evaluate the Transformer-based Neural Machine Translation model.

## Dataset Link

https://drive.google.com/file/d/1mSHfcKABnUWStgdsq6Z74gq7cw9mt6Ev/view?usp=drivesdk

https://drive.google.com/file/d/1uwX-Ue81hrynGVD5aVXk_veiqDRKL42J/view?usp=drivesdk

## Dataset Contents

- `train.en` – English sentences
- `train.hi` – Hindi translations

## Usage

1. Download the dataset from the link above.
2. Extract the files (if compressed).
3. Place the dataset files in the project's data directory.
4. Run the preprocessing and training scripts as described in the project documentation.

> **Note:** The dataset is provided for educational and research purposes. Please refer to the original dataset source for licensing and usage terms, if applicable.
